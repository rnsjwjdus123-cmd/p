Santokki Strategy API

This backend serves the analysis API used by the Santokki strategy and dashboard pages.

Run the main Santokki site (from the p folder) and open:

  • strategy.html — Brand Success Strategy (Risk · Compliance · Competitors · ROI)
  • unified-dashboard.html — Firebase stats, Prompt generator, Influencer discovery, Product strategy

In those pages, set Analysis API URL to this server (e.g. http://localhost:8000).
