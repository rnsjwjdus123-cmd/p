## 📋 회의 의제 (기획자)

현재 상황:
- 자체 코드로 Instagram Webhook + DM 자동화 구현 완료
- 그러나 Meta 앱 검수에 **비즈니스 인증(법인 등록 필요)** → 개인 사업자/법인 없으면 통과 불가
- 코드는 100% 작동하지만 **live mode 전환이 안 됨**

**핵심 논점:**
1. 법인 없이 동일한 자동화를 실현할 수 있는가?
2. 기존 코드 자산을 얼마나 활용할 수 있는가?
3. 비용 대비 효과가 가장 좋은 방법은?
4. 향후 확장성(Phase 2)을 고려한 최적 방법은?

## 📊 비즈니스 분석 (비즈니스 분석가)

### 대안 분류 — 3가지 경로

| 경로 | 방법 | 법인 필요 | 비용 | 구현 난이도 | Phase 2 확장성 |
|------|------|----------|------|-----------|-------------|
| **A** | ManyChat (노코드 SaaS) | ❌ 불필요 | 무료~$15/월 | ⭐ 쉬움 | ⭐⭐⭐ |
| **B** | Inrō / ChatAutoDM | ❌ 불필요 | $20~40/월 | ⭐ 쉬움 | ⭐⭐ |
| **C** | 법인 설립 후 자체 코드 | ✅ 필요 | 법인 설립비 + 서버비 | ⭐⭐⭐ 어려움 | ⭐⭐⭐⭐⭐ |

### 경로 A: ManyChat (가장 추천 ⭐)

**장점:**
- **Meta 공식 파트너** → 앱 검수 불필요! (ManyChat이 대신 처리)
- **무료 플랜**: 월 1,000 contacts까지 무료
- **댓글 → DM 자동화** 바로 사용 가능 (Comment Growth Tool)
- **노코드 플로우 빌더**로 시향테스트 퀴즈도 DM 내에서 구현 가능
- Instagram 비즈니스 계정만 있으면 5분 만에 셋업

**단점:**
- 복잡한 커스텀 로직은 제한적
- Pro 플랜부터 월 $15 (1,000명 초과 시)
- 데이터가 ManyChat 서버에 저장됨

### 경로 B: Inrō (Instagram 특화)

**장점:**
- 댓글 → DM 캠페인에 특화
- AI 기반 자동 응답
- 활성 컨택트 기준 과금 (효율적)

**단점:**
- ManyChat 대비 기능 제한
- 한국어 지원 부족할 수 있음

### 경로 C: 법인 설립 후 자체 코드

**장점:**
- 완전한 커스텀 (이미 코드 구현 완료)
- 데이터 100% 자사 관리
- 무한 확장 가능

**단점:**
- 법인 설립 비용 + 시간 (최소 1~2주)
- Meta 앱 검수 추가 1~2주
- 서버 운영 비용 (월 $5~20)

## 💡 크리에이티브 제안 (크리에이티브 디렉터)

### 🌟 하이브리드 전략 (단기 + 장기)

**즉시 실행 (이번 주):** ManyChat으로 Phase 1 바로 가동
- 댓글 "시향테스트" 감지 → DM으로 시향테스트 링크 발송
- 무료 플랜으로 시작, 반응 확인

**2주 후:** ManyChat에서 Phase 2 구현
- DM 내에서 Quick Reply 기반 시향테스트 퀴즈 직접 구현 (외부 링크 불필요!)
- 답변 기반 제품 추천 메시지 자동 발송
- 프로모코드 자동 발급
- 구독자 태깅 → 월별 할인 알림 자동화

**장기 (법인 설립 후):** 자체 코드 + ManyChat API 연동
- ManyChat API로 기존 Python 코드와 연결
- 더 고도화된 매칭 알고리즘 적용
- 데이터 자사 DB 이관

### 🎯 ManyChat에서 시향테스트 구현 방법

```
[댓글: "시향테스트"]
    ↓ (자동 DM)
"안녕하세요! 🐰 시향 테스트를 시작할게요!"
"어떤 분위기를 좋아하세요?"
    [🌲 자연/숲] [🌸 꽃/봄] [🍋 상쾌/시트러스]
    ↓ (Quick Reply 선택)
"어디서 주로 향을 사용하세요?"
    [🛋️ 거실] [🛏️ 침실] [🚗 차량]
    ↓
"원하는 향의 강도는?"
    [은은하게] [적당히] [강하게]
    ↓
"당신에게 딱 맞는 향은... ✨수풀 룸 스프레이✨"
"👉 구매하기: [링크]"
"🎁 첫 구매 15% 할인: SANTOKKI-XXXX"
```

이렇게 하면 **외부 설문 링크도 필요 없이** DM 안에서 모든 과정이 완료됩니다!
