## 1. 기능 설명 (전체)

### 1.1 쇼핑몰 (메인)
| 기능 | 설명 |
|------|------|
| **메인 페이지** | index.html — 한국 홈 프래그런스 브랜드 Santokki 소개, 상품 진입 |
| **상품 목록·상세** | product.html, product-detail.js, products.json(14종) |
| **장바구니·결제·인증** | cart.html, checkout.html, auth.html |
| **스타일·공통 스크립트** | styles.css, script.js |

### 1.2 Hub (진입점)
| 기능 | 설명 |
|------|------|
| **Hub** | hub.html — Strategy / Dashboard 선택, 구글 시트 링크(API 연동 시 표시) |
| **Strategy** | strategy.html — 브랜드 성공 전략(리스크·컴플라이언스·경쟁사 포지셔닝·ROI 시뮬레이션). 고정 시드(SANTOKKI_BENCHMARK) + Run analysis(Python API) |
| **Dashboard** | unified-dashboard.html — 통합 대시보드(인플루언서 발견·제품 전략·Run analysis). embed=1 로 iframe 삽입 가능 |

### 1.3 프롬프트 생성기
| 기능 | 설명 |
|------|------|
| **페이지** | prompt-generator.html |
| **Firebase 통계로 생성** | 시향 테스트 10명 이상 시 통계 수집 → 최적 상품·15초 광고·게시글·SEO 생성 |
| **직접 입력으로 생성** | 인기 향·제품·응답 수 입력 후 생성 |
| **예쁘게 만들기** | 인스타·블로그용 포맷 (format-for-social API) |
| **시트 저장** | 생성 결과를 구글 시트에 추가 (GOOGLE_SHEET_ID 등 설정 시) |
| **config 연동** | config.json의 apiBase로 Node API 주소 지정(배포 시 Render URL) |

### 1.4 API 서버 (Node — server.js)
| API | 설명 |
|-----|------|
| POST /api/generate-prompts | 프롬프트 생성 (OpenAI) |
| GET /api/stats-from-firebase | Firebase 시향 테스트 통계 |
| POST /api/save-prompt-to-sheet | 구글 시트에 행 추가 |
| POST /api/format-for-social | 인스타·블로그용 포맷 |
| GET /api/sheet-url | 구글 시트 설정 여부·URL (Hub·프롬프트 생성기에서 사용) |
| POST /api/manychat/webhook | ManyChat 웹훅 수신 (퍼널 카운트) |
| GET /api/funnel-stats | ManyChat 퍼널 숫자 (Firebase) |
| POST /api/run-daily-prompts | 일일 프롬프트 → 시트 자동 저장 (cron/수동) |

### 1.5 Strategy/Dashboard 분석 API (Python — project_html/backend)
| API | 설명 |
|-----|------|
| GET /api/products | 제품 목록 |
| POST /api/analyze | 제품별 분석 (OpenAI + Perplexity): 타겟 페르소나, UK 문화 적합성, 경쟁사, ROI 시뮬레이션, 인플루언서 추천 등 |
| **환경 변수** | OPENAI_API_KEY, PERPLEXITY_API_KEY |
| **config 연동** | config.json의 strategyApiBase로 배포 시 Render URL 지정 |

### 1.6 시향 퀴즈·인스타 자동화 (santokki/)
| 기능 | 설명 |
|------|------|
| **시향 퀴즈** | santokki/quiz/ — 15문항 퀴즈, Firebase Hosting 배포 |
| **대시보드** | santokki/dashboard.html — 시향 테스트 통계, ManyChat 퍼널 |
| **n8n** | santokki/n8n-docker — DM 자동화 워크플로우 |
| **ManyChat** | 댓글 키워드 → 퀴즈 링크 DM 등 (server.js 웹훅 수신) |

### 1.7 배포 구성
| 대상 | 서비스 | 비고 |
|------|--------|------|
| 정적 페이지 | Netlify | 메인, Hub, Strategy, Dashboard, 프롬프트 생성기 |
| Node API | Render (api-santokki-2835) | 프롬프트 생성·시트·Hub 시트 링크 |
| Python API | Render (p-1) | Strategy/Dashboard Run analysis |
| config.json | apiBase, strategyApiBase | Netlify 배포본이 두 API 주소 참조 |
